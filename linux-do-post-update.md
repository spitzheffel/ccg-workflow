## 🌐 Web UI 功能（v1.7.3 新增）

### ✨ 自动打开浏览器

执行命令时，系统会：
1. 自动打开浏览器标签页
2. 显示实时输出（SSE 流式推送）
3. 任务完成后显示 "✓ 完成"

### 🎨 内容类型显示

Web UI 会根据内容类型使用不同样式：

| 类型 | 图标 | 样式 | 说明 |
|------|------|------|------|
| **思考过程** | 💭 | 灰色斜体 | Codex/Gemini 的推理过程 |
| **命令执行** | 💻 | 黄色代码块 | 执行的命令和输出 |
| **最终回复** | 📝 | 白色正常 | 生成的代码或文本 |

### 📊 流式对比

| 后端 | 流式效果 | 说明 |
|------|----------|------|
| **Gemini** | ✅ 真正流式 | 逐字逐句推送 |
| **Codex** | ⚡ 分块输出 | 思考→命令→回复分开显示 |
